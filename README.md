# random-lotr-movie-quote

> Get a random The Lord of the Rings movie quote.

## Install

```bash
npm install random-lotr-movie-quote
```

## Usage

```javascript
// code away!

const randomQuote = require('random-lotr-movie-quote');
console.log(randomQuote());
```
